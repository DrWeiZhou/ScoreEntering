# 按照以下操作步骤执行，如果操作中有截图，将截图图片也保存在项目根目录中：
1. 阅读D:\WEIZHOU_QTECH_2309\WEIZHOU_NOTE\Obsidian\WORK_NOTE\工作\2025-2026-1\网安课设答辩.md文件，读取其中## 数据结构 节中的内容，获取每行中学生的姓名（2-3个汉字）及其前空格前的成绩，输出在控制台中，并让我确认下是否正确
2. 打开网页https://docs.qq.com/sheet/DTVdaUUNOSFp4TVdI?rtkey=eb6376f753d30c5baf7bed8bya8iv1&tab=BB08J2，出现标题为“数据结构课程设计成绩”的页面即可

4. 解析下载的xlsx文件中的D列（姓名列）找到md中每个学生所在的行数
5. 使用按键操作对在线文档进行操作，刚打开页面时，第1行A列的单元格被选中，移动到对应行数，读取D列(标题为"姓名"）列中的名字，如果和md中学生名字一样，填写该行中的以下列的分值（仅填写一个整数）：
- E列（最大15分）
- F列（最大15分）
- G列（最大20分）
- H列（最大20分）
- I列（最大10分）
- J列（最大20分）
使得E到J列所有列的整数之和等于MD文件中的成绩
